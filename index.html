¿Que aprenderás?

Un desarrollador web debe dominar 4 principales lenguajes

HTML 5

Crearás la estructura de aplicaciones compatibles con todos los navegadores. Con HTML5 le decimos al navegador lo que vamos a visualizar como barras de Navegación, Secciones, artículos, listas, títulos, audios, videos, imágenes, entre otros.

CSS3

Es el lenguaje estandar con el que daremos el estilo a todo el documento HTML5. Con el diseñaremos un aspecto profesional a nuestra aplicación web, logrando un diseño actual, fácil de utilizar y agradable a la vista para los usuarios.

JQUERY

Es un framework de Javascrip que se emplea para brindar a las aplicaciones la capacidad de interactuar con los usuario de una forma dinámica, rápida y fácil agregando efectos visuales y optimizar el envio de informacion a servidores y base de datos.

PHP

Es el lenguaje por excelencia que todo programador web debe dominar, es el lenguaje con la comunidad mas grande a nivel mundial. PHP nos servirá para poder guardar, editar y eliminar informacion en la base de datos

Ver temario de los modulos
Modulo 1: 6 meses

1. Desarrollo Frontend

Aprende a diseñar interfaces de usuario de las aplicaciones con HTML5 , CSS3 y javascript adaptables a diferentes pantallas de dispositivos móviles.

Modulo 2: 6 meses

2. Desarrollo Backend

Aprende a programar tu aplicación con PHP y MariaDB, crear tu base de datos y conectarla con tu aplicación para guardar, editar y eliminar información.

Especialización: 6 meses

3. Bootstrap 5

Aprende Bootstrap, Framework Diseñado para facilitar el proceso de desarrollo de los sitios web responsivos y orientados a los dispositivos móviles.
